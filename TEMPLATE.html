Machen Example

Welcome to Machen

Machen is a simple framework I'm intending to use as a revision tool.

Slides are easy to define. Simply create a div inside the #content div and add the class "machen-panel".

The tool uses a jquery plugin I made called jquery.machen to do the navigation through the slides. jquery.machen is therefore able to be used in other projects where you need to provide movement through different slides.

Navigation and Mode keys:

  • Use the left or right keys to move back and forwards through the slides
  • Use the Enter or Spacebar keys to move to the next slide
  • Use the Backspace key to move back a slide
  • Click the main container area to move forward a slide
  • Press 'f' to toggle full browser window mode (this hides or shows the top and bottom bars
  • You can also navigat using the links at the top left of the page.

You can also add an answer div inside each "machen-panel" by adding the div and assigning the class "machen-answer" to the div. When you navigation forward the framework will check to see if there is an answer div and show it if it has not been shown.
